openin by solenskiner

An application for opening links from other applications, if you use multiple browsers. Pops up a dialog to choose browser.

Set it as default browser in your Desktop Environment, or if you have none, export BROWSER=/path-to-executable/openin in your .xinitrc or .xprofile.

Currently supports: firefox, chrome/chromium and opera. Easely extendable; just edit findbrowser.cpp and recompile.
